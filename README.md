// What is the difference between var, let, and const?
ans :- var is the old way to declare variables and can be reassigned or redeclared, which may cause confusion.
let is a modern version that can be reassigned but not redeclared in the same scope.
const is used when the value should not change after being assigned.

// What is the spread operator (...)?
ans :- The spread operator (`...`) is used to copy or expand values from an array or object into another array, object, or function.
It makes code shorter and easier to read, especially when combining or copying data.

// What is the difference between map(), filter(), and forEach()?
ans :- `map()` creates a new array by changing each item, `filter()` creates a new array with only the items that match a condition, and `forEach()` simply loops through items without returning a new array.
They all work on arrays, but each one is used for a different purpose.

// What is an arrow function?
ans :- An arrow function is a shorter and cleaner way to write a function in JavaScript using `=>`.
It helps make the code easier to read, especially for small functions.

// What are template literals?
ans :- Template literals are a modern way to write strings in JavaScript using backticks (`` ` ` ``) instead of quotes.
They make it easy to add variables or write multi-line text inside a string.
